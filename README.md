# Guessing_Number_Simple_Game
- A sleek, interactive number guessing game with modern glassmorphism design and smooth animations.
- **Live Demo:** [https://redluzon.github.io/Guessing_Number_Simple_Game/guessing_number_game.html]

### Features
- Custom difficulty range (set your own max number)
- 3 attempts with hint system (Too High/Too Low)
- Confetti celebration on win
- Shake animation for wrong guesses
- Responsive design for all devices
- Landing page with game instructions
- <img width="1862" height="990" alt="image" src="https://github.com/user-attachments/assets/a78b61e9-dbe5-4ff6-858e-b5e55db713a6" />


### Tech Stack
- HTML5, CSS3, JavaScript
- Google Fonts (Outfit)
- Font Awesome icons

### How to Play
- Enter a maximum number to set difficulty
- Guess the secret number in 3 tries
- Use hints to narrow down your next guess
- Win and enjoy the confetti!
